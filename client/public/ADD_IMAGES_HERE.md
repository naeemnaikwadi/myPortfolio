## Required Images

Place these images in this folder (`client/public/`):

### Personal Photos
- `profile.jpg` - Your professional headshot (500x500px)
- `workspace.jpg` - Your coding setup (1200x800px)
- `resume.pdf` - Your resume PDF

### Project Screenshots (in `projects/` subfolder)
Create a `projects/` folder here and add:
- `growskill.jpg` - GrowSkill.edu screenshot
- `skillsync.jpg` - SkillSync screenshot
- `revcorp.jpg` - REVCORP screenshot
- `icam.jpg` - iCam screenshot

## Quick Setup

```bash
# Create projects folder
mkdir client/public/projects

# Add your images
# Copy your images to:
# - client/public/profile.jpg

```

## Image Specifications

### profile.jpg
- Size: 500x500px (square)
- Format: JPG or PNG
- Max file size: 200KB
- Style: Professional headshot

### workspace.jpg
- Size: 1200x800px (landscape)
- Format: JPG
- Max file size: 300KB
- Style: Your coding setup

### Project Screenshots
- Size: 1200x800px (landscape)
- Format: JPG or PNG
- Max file size: 400KB each
- Style: Clean UI screenshots

## Temporary Solution

Until you add real images, the portfolio will show:
- Gradient backgrounds for projects
- Placeholder areas

The site will still work perfectly!

## After Adding Images

1. Add images to this folder
2. Commit to Git:
   ```bash
   git add client/public/
   git commit -m "Add portfolio images"
   git push
   ```
3. Vercel will auto-deploy
4. Images will appear on your site!
