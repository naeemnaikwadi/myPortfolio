# Image Requirements

Place the following images in this directory:

## Required Images

### Profile & Personal
- **profile.jpg** (500x500px, square)
  - Professional headshot
  - Clear background
  - Front-facing, smiling
  - High resolution

- **workspace.jpg** (1200x800px, landscape)
  - Your coding setup
  - Clean and professional
  - Good lighting

### Projects (in /projects folder)
- **growskill.jpg** (1200x800px)
  - Screenshot of GrowSkill.edu dashboard or app
  - Show key features
  
- **skillsync.jpg** (1200x800px)
  - SkillSync dashboard or login interface
  - Highlight main functionality

- **revcorp.jpg** (1200x800px)
  - REVCORP review submission or UI
  - Show the platform in action

- **icam.jpg** (600x1200px, mobile)
  - iCam mobile app screenshot
  - Object detection in action

### Optional
- **contact-illustration.svg**
  - Mailbox or communication themed illustration
  - Can download from: undraw.co, storyset.com

## Image Sources

Free stock photos:
- Unsplash.com
- Pexels.com
- Pixabay.com

Free illustrations:
- undraw.co
- storyset.com
- freepik.com

## Tips

1. Optimize images before uploading (use TinyPNG.com)
2. Use consistent aspect ratios
3. Ensure good contrast for dark mode
4. Keep file sizes under 500KB each
5. Use descriptive alt text in components
